ğŸ“– About The Project

AlgoViz is a modern, high-performance visualization playground designed to make learning computer science algorithms intuitive and engaging.

Built with React, Vite, and Framer Motion, it moves away from clunky, old-school visualizers, offering instead a smooth, "neon cyber" aesthetic. It visually explains fundamental sorting and searching algorithms through animated elements, step-by-step comparisons, and cinematic highlights.

Target Audience: Students, educators, and developers looking for a clean, browser-native educational tool.
